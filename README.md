# Web Scraper & File Monitor Projects

Этот репозиторий содержит два полезных инструмента для автоматизации работы с веб-контентом и файловой системой.

## 📁 Проекты

### 1. Python Web Scraper (`header_scraper.py`)
Веб-скрапер на Python, который собирает заголовки (h1-h6) с указанного сайта и сохраняет их в CSV-файл.

**Особенности:**
- Сбор всех заголовков h1-h6 со страницы
- Сохранение результатов в CSV-файл
- Поддержка аргументов командной строки
- Обработка ошибок и таймаутов
- Работа с сайтами, имеющими самоподписанные SSL-сертификаты

**Установка зависимостей:**
```bash
pip install requests beautifulsoup4
```

**Использование:**
```bash
python header_scraper.py https://example.com
python header_scraper.py https://example.com -o my_headers.csv
python header_scraper.py --help
```

**Выходной файл:**
Результаты сохраняются в CSV-файл с колонками: `tag`, `text`, `level`.

---

### 2. Node.js File Monitor (`monitor.js`)
Скрипт на Node.js для мониторинга изменений в папке и автоматической обработки новых файлов.

**Особенности:**
- Отслеживание появления новых файлов в реальном времени
- Игнорирование временных файлов
- Автоматическая обработка новых файлов
- Гибкая настройка пути к папке

**Требования:**
- Node.js (встроенные модули, дополнительные зависимости не требуются)

**Использование:**
1. Откройте `monitor.js` и измените путь к папке в начале файла:
   ```javascript
   const WATCH_PATH = './watched_folder';
   ```
2. Запустите скрипт:
   ```bash
   node monitor.js
   ```
3. Добавьте новые файлы в отслеживаемую папку для проверки работы.
